CSS Flexbox

CSS flexbox (also referred to as flexible box) provides web authors with control over how elements are positioned, aligned, and sized within their container. It allows you to do things like, specify how elements are aligned vertically and horizontally, change their order or appearance, change the direction in which all elements are laid out, and more.
